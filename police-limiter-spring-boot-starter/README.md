# 基于注解的MOCK数据的AOP


## 快速开始

### 引入pom

```xml
<dependency>
    <groupId>com.lhsj</groupId>
    <artifactId>police-limiter-spring-boot-starter</artifactId>
    <version>xxxx</version>
</dependency>
```

### 配置properties

表示启用此组件

```properties
# mock
police.limiter.enable=true
```

### 使用

在方法上，写上注解即可。
